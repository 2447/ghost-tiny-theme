## Feature

* 仅有博文列表以及博文内容两个页面，极简
* 代码高亮为vue风格
* 响应式

## How to use

[下载主题压缩包](https://github.com/zhaohaodang/tiny/archive/1.0.0.zip)，将解压后tiny-x.x.x文件夹中的tiny-x.x.x文件夹复制到`Ghost-x.x.x\content\themes`目录下，重新启动Ghost。
